OpenAuth is a new authentication provider built on OAuth 2.0 that can be deployed standalone or integrated into existing apps. It works across any framework or platform, runs on your own infrastructure (Node.js, Bun, AWS Lambda, Cloudflare Workers), follows web standards, and includes customizable UI components.
